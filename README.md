#Лабораторная работа: Знакомство с системой контроля версий Git

##В этой лабораторной работе необходимо освоить базовые команды Git, работу с ветками, коммитами и разрешением конфликтов.

##📌 Задания
###1. Работа с основной веткой (main)

Добавить строку "экзамен" в файл data.txt и выполнить коммит.

Изменить файл data.txt, добавить строку "не экзамен" и сделать коммит.

###2. Работа с ветками

Перейти на ветку hotfix, добавить строку "экзамен" в data.txt и сделать коммит.

Посмотреть разницу между последними двумя коммитами и сохранить вывод в файл diff.txt.

###3. Работа с откатами и конфликтами

Откатиться к третьему коммиту в ветке main, изменить data.txt, добавив строку "экзамен", и сделать новый коммит.

Перейти на ветку hotfix и выполнить слияние (merge) из main.
Разрешить возможные конфликты.

###4. Работа с несколькими ветками

Слить ветки feature1 и feature2 в main.
Если возникнут конфликты — разрешить их вручную и выполнить merge-коммит.

Создать ветку release от main и выполнить слияние всех изменений.

###5. Работа с GitHub

Создать аккаунт на GitHub и оформить репозиторий:

заполнить bio

изменить аватар

создать файл README.md

выполнить базовую кастомизацию профиля
